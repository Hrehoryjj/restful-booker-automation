# Test Plan: Restful Booker Automation

## 1. Introduction
This project aims to automate testing for the Restful Booker platform, covering both API endpoints and UI elements.

## 2. Testing Levels
- **API Testing:** Focus on business logic, status codes, and data integrity.
- **UI Testing:** Focus on critical user journeys on the website.
- **E2E Testing:** Cross-layer scenarios (e.g., create booking via API and verify in UI).

## 3. Technology Stack
- **Language:** Java 17
- **UI:** Selenium WebDriver
- **API:** RestAssured
- **Framework:** TestNG
- **Reporting:** Allure
